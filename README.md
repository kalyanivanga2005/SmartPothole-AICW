# 🚧 Smart Pothole Detection and Reporting System

## 📌 Project Overview

The Smart Pothole Detection and Reporting System is designed to improve road safety by automatically detecting potholes and notifying users and authorities in real time. The system uses mobile sensors, GPS technology, cloud storage, and voice alerts to identify potholes and provide location-based warnings.

This project aims to reduce road accidents, vehicle damage, and maintenance delays through an intelligent and automated reporting mechanism.

---

## 🎯 Objectives

- Detect potholes automatically using mobile sensors.
- Identify pothole locations using GPS.
- Store pothole information in Firebase.
- Alert drivers through voice notifications.
- Help authorities monitor and repair damaged roads quickly.

## ⚙ System Workflow

1. Vehicle moves on the road.
2. Mobile sensors detect road vibrations.
3. GPS captures the current location.
4. Pothole information is uploaded to Firebase.
5. Distance between user and pothole is calculated.
6. If the pothole is within approximately 100 meters, a voice alert is generated.
7. Authorities can access stored pothole data for maintenance.
